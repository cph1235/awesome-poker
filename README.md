# awesome-poker

Dependencies:
1. ReactJS: https://facebook.github.io/react/
2. Flask: http://flask.pocoo.org/
3. Sqlite: https://www.tutorialspoint.com/sqlite/
4. Webpack: https://webpack.github.io/

Prerequisites:
1. npm
2. pip

Steps to setup:
1. run pip install flask
2. run npm install -g webpack
3. run npm install
4. run npm build
5. run npm start
